# 婚禮 Checklist v3

本版更新：
- 移除重複的首頁，開啟後直接顯示全部任務
- 移除期限／到期時間
- 保留工作內容與準備物品
- 任務、人員、分類皆可新增、修改、刪除
- 修正「取消」後視窗不關閉與內容殘留
- 點擊視窗外背景也可關閉
- Firebase 多裝置即時同步

## 更新方式
把 ZIP 內 5 個檔案上傳到 GitHub Repository 根目錄並覆蓋舊檔：
- index.html
- app.js
- firebase-config.js
- firestore.rules
- README.md

這版 Firestore 規則與 v2 相同；若已發布 v2 規則，不必再次更新 Firebase。
