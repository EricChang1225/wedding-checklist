# 婚禮管家 2.2｜我的行程簡化版

## 我的行程
取消原本獨立的「我的名單」區塊。

首頁現在只顯示：

1. 下一個行程
2. 今天要做
3. 今天要帶
4. 我的行程表

「今天要做」會自動整合：
- 婚禮準備中的工作
- 工作名單中的個人工作
- 流程確認項目

例如只要把燈叔加入：
- 喝茶名單
- 第 6 位
- 關聯到文定流程

燈叔的「我的行程」就會自動出現：
- 文定流程
- 第 6 位喝茶
- 集合時間與地點

不用再到其他地方重複維護。

## 工作名單
仍保留給管理者維護：
- 喝茶名單
- 招待名單
- 收禮金
- 送客
- 伴郎伴娘等

## 管理總覽
管理模式新增「總覽」，顯示：
- 前置完成率
- 流程確認率
- 名單數量與成員數
- 待處理數量
- 各婚禮階段狀態

## Firebase
沿用 2.1 的：
- wccRosters
- wccRosterMembers

如果已經發布過 2.1 的 firestore.rules，本版不用再更新規則。
