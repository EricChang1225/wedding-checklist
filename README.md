# Wedding Control Center v11（單一網站整合版）

## 這版解決的問題
之後只需要開一個網址：

https://ericchang1225.github.io/wedding-checklist/

主網站新增「婚宴中心」分頁，點進去即可使用：

- 婚宴桌位與賓客搜尋
- 已匯入的桌次資料
- 婚宴桌圖
- 婚宴菜單
- 協同廠商
- Excel 重新匯入
- CSV 匯出

不需要再另外開 banquet.html 網址。

## 更新方式
將 ZIP 中所有檔案上傳到 GitHub 根目錄並覆蓋舊檔。

## Firebase
本版沿用 v10 的 Firestore collections 與規則。
如果 Firebase 已發布 v10 的 firestore.rules，這次不用再發布。
