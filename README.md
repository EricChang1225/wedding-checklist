# 婚禮管家 V3.7｜婚禮流程連結工作名單版

## 新增功能
新增或修改婚禮流程時，可複選既有的工作名單。

例如「11:00 婚宴」可連結：
- 招待名單
- 陪嫁名單
- 收禮名單

## 婚禮流程顯示
流程展開後會顯示：
- 已連結的工作名單
- 每份名單的人數
- 名單成員
- 個人工作內容

也可按「前往工作名單」進入完整管理頁。

## 7/18 即時行程
目前或下一個流程若有連結工作名單，會直接顯示名單名稱。

## 資料方式
工作名單仍只建立一次。
婚禮流程只儲存名單連結，不會複製名單或成員資料，因此修改工作名單後，流程畫面會同步更新。
