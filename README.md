# 婚禮管家 V5.2｜總招待預覽版

## 設定方式
到「工作名單」修改「招待名單」，新增欄位：

- 總招待：選擇爐叔

爐叔不必加入9位招待成員中，因此名單仍維持9人。

## 爐叔的我的行程
當目前使用者選擇爐叔時，會自動出現「我的招待」卡片，內容包括：

- 招待名單名稱
- 招待總人數
- 每位招待姓名
- 個人工作
- 負責桌次
- 點桌號直接查看該桌賓客資料

## 其他使用者
不是該名單總招待的人，不會看到「我的招待」卡片。

將 ZIP 內全部檔案覆蓋上傳 GitHub Repository 根目錄。
