# 婚禮管家 V3.7.2｜婚禮流程上下移動修正版

## 修正內容
婚禮流程原本雖然有更新 `sort` 欄位，但畫面仍然固定依時間排序，所以按「上移／下移」看起來完全沒有作用。

這版已修正：

- 上移與下移會真正改變同一階段內的流程順序
- 每次移動後會重新寫入完整且連續的排序值
- 可避免舊資料排序值重複或跳號造成失效
- 重新整理後仍維持調整後的順序
- 不會修改流程原本的時間、負責人或工作名單

## 注意
上下移動只會在同一個流程階段內調整，不會把流程移到其他階段。
