# Wedding Control Center 1.0 正式版（v12）

## 管理模式
- 專案尚未設定管理密碼時，只會由第一位開啟的人建立一次。
- 密碼設定完成後，其他人不會再看到建立密碼畫面。
- 一般模式可以查看、搜尋、開地圖，以及勾選工作／細項／流程確認。
- 新增、修改、刪除、排序與設定只在管理模式顯示。
- 點右上角 🔒，或長按網站標題約 2 秒，可輸入管理密碼。
- 管理模式 30 分鐘後自動失效，也可在設定中主動退出。
- 密碼以 SHA-256 雜湊保存，不會直接儲存原始密碼。

## 婚禮流程準備細項
流程中連結前置項目時，會顯示：
- 已準備／尚未準備
- 完成數量
- 可展開的全部準備細項
- 可直接勾選細項並同步回前置準備

## 婚宴中心
婚宴菜單、協同廠商、桌圖設定與 Excel 匯入同樣受到管理模式控制。
桌位查詢、搜尋與查看名單維持所有人可用。

## 安全說明
目前管理密碼主要用來避免家人或工作人員誤改資料。因為網站仍是純前端 GitHub Pages，Firestore 規則無法驗證這組前端密碼；它不是防止專業攻擊者的高強度登入機制。
