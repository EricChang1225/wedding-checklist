# 婚禮 Checklist v4

## 這版重點
- 負責人可自由輸入
- 可加入多位負責人
- 新姓名會自動加入「人員」
- 已建立的人員會顯示為建議
- 不再預設新郎、新娘、伴郎、伴娘、主持人
- 任務、人員、分類皆可新增／修改／刪除
- 全部任務為預設首頁
- 移除到期日期
- 取消按鈕可正常關閉視窗
- 點背景也可關閉視窗
- Firebase 即時同步

## GitHub 更新方式
把以下檔案全部上傳至 Repository 根目錄，覆蓋舊檔：
- index.html
- style.css
- app.js
- firebase-config.js
- firestore.rules
- README.md

## Firebase
firestore.rules 與前一版相同。若你已經發布過相同規則，不必再更新。
