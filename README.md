# 婚禮管家 V3.0.15｜婚禮流程多人負責人版

## 婚禮流程負責人
婚禮流程的「負責人」已改成與人員頁同步的可複選清單。

可以同時選擇多位負責人，例如：
- 松叔
- 紹廷
- 攝影師

## 自動連動
被指定為流程負責人的人，會在自己的「我的行程」看到該流程。

## 相容既有資料
既有流程中原本的單一負責人不需要重建。
系統會同時支援：
- 舊欄位 owner
- 新欄位 owners

## 顯示
婚禮流程、我的行程與任務中心都會顯示完整負責人名單。
