# 婚禮管家 V3.0.9｜婚禮準備 List 版

## 婚禮準備新介面
婚禮準備改為單一清單，一目了然顯示：
- 完成狀態
- 準備內容
- 誰準備
- 大項
- 細項進度
- 操作

## 大項篩選
畫面上方提供：
- 全部
- 文定
- 迎娶
- 婚宴
- 以及所有自行新增的大項

「＋新增大項」可自由建立拜別、證婚、送客等分類。

## 與我的行程連動
新增或修改準備內容時，只要指定負責人：
- 該項目會自動出現在該使用者的「我的行程 → 準備物品」
- 切換使用者後，首頁會顯示對應人員的內容
- 資料只建立一次，不會重複

## 細項
每列可：
- 加細項
- 展開／收合細項
- 修改
- 刪除
- 勾選完成
