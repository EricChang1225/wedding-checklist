# Wedding Control Center v16：我的任務工作台

## 修正
婚禮流程中設定的負責人，現在會出現在「我的任務」。

## 我的任務分成四區

1. 我的前置工作
2. 我的準備物品
3. 我的婚禮流程
4. 我的流程確認項目

頁面最上方會顯示各類未完成數量。

## 接下來 1 小時
若目前使用者負責的流程有設定單一時間或開始時間，且距離現在 0～60 分鐘，會顯示在「接下來 1 小時」。

## 前往流程
在個人頁按「前往流程」，系統會：
- 自動切換到婚禮流程
- 展開所屬婚禮階段
- 展開該流程
- 捲動到該流程附近

## 使用注意
「目前使用者」名稱必須和流程／工作內填寫的負責人完全相同。

例如目前使用者是：
新郎家

流程負責人也必須填：
新郎家

若一邊寫「新郎家」、另一邊寫「新郎」，系統會視為不同的人。

沒有新增 collection，不需要修改或重新發布 firestore.rules。
