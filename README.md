# 婚禮管家 V3.0.12｜婚禮準備大項管理版

## 新增功能
在「婚禮準備」選取某個大項後，右側會出現：

- 上移
- 下移
- 修改大項
- 刪除大項

## 刪除規則
- 大項沒有準備內容：確認後直接刪除。
- 大項仍有準備內容：會先顯示筆數，確認後連同該大項下的準備內容、細項，以及由這些內容產生的流程確認項一起刪除。
- 刪除完成後，畫面會自動切回「全部」。

## 修改大項
可以修改：
- 大項名稱
- 圖示
