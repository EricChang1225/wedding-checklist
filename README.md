# 婚禮管家 V3.4｜招待負責桌次版

## 工作名單新增「負責桌次」
在工作名單加入或修改成員時，可複選該招待負責的桌次。

桌次來源直接使用既有的婚宴桌次資料，不需要重新建立。

## 工作名單顯示
被指派的桌次會顯示在該成員下方，例如：

- 1桌
- 2桌
- 8桌

點桌號可開啟婚宴桌次頁。

## 我的行程
目前使用者若被工作名單指派桌次，「我的行程」會新增：

- 我的負責桌次
- 桌號
- 桌名
- 所屬工作名單
- 個人工作內容

點桌次卡片可直接開啟婚宴資訊頁。
