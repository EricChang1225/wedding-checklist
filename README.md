# 婚禮管家 V3.2｜婚禮流程時間軸＋我的任務排序版

## 婚禮流程
婚禮流程改成真正的時間軸：

- 左側顯示時間
- 右側顯示流程卡片
- 預設只顯示流程名稱、地點、負責人與完成進度
- 點流程卡片才展開地圖、備註與確認項目
- 沒有內容時不再顯示「沒有細項」
- 管理按鈕集中在展開區，普通使用者畫面更乾淨

## 我的行程排序
「我的行程」中的任務會依下列順序排列：

1. 任務開始時間
2. 若沒有任務時間，使用所連結流程的時間
3. 完全沒有時間的任務排在最後

準備物品也使用相同排序邏輯。

## 保留功能
- 任務中心
- 多位負責人
- 婚禮準備 PDF 工作清單
- Header 精簡
- 移除總覽
