# 婚禮管家 3.0｜獨立工作中心版

## 本版重點

### 倒數移到 Header
「距離婚禮還有幾天」已整合到上方婚禮管家標題區。
首頁不再另外占用一張大型倒數卡片。

### 新增工作中心
婚禮流程只保留正式婚禮節點，例如：
- 文定
- 迎娶
- 婚宴
- 送客

以下額外工作改放在「工作中心」：
- 去接主持人
- 去載長輩
- 拿蛋糕
- 買冰塊
- 搬飲料
- 臨時支援

獨立工作可以設定：
- 負責人
- 工作分類
- 日期
- 開始與結束時間
- 地點與 Google 導航
- 重要程度
- 可選擇是否關聯婚禮流程

即使沒有關聯流程，只要負責人與目前使用者相同，就會顯示在「我的行程 → 今天要做」。

### 準備物品獨立
「準備物品」分頁只顯示物品與準備包，不再和獨立工作混在一起。

## 資料相容
沿用現有 wccTasks collection，只新增可選欄位，不會刪除既有資料。

## Firebase
沒有新增 collection。
已發布 2.1 規則者，不需要重新更新 firestore.rules。
