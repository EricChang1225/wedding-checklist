# 婚禮管家 V4｜一般模式精簡導覽版

## 導覽順序

### 一般模式
只顯示：

1. 我的行程
2. 婚禮流程
3. 工作名單
4. 婚宴資訊

「婚禮流程」已放在「我的行程」旁邊。

### 管理模式
進入管理模式後，會在同一排追加：

5. 任務中心
6. 婚禮準備
7. 人員
8. 設定

## 權限行為
- 一般模式完全看不到管理頁籤
- 管理模式退出後，管理頁籤會立即消失
- 若正在管理頁面時退出管理模式，會自動返回「我的行程」
- 非管理者無法用程式或舊連結直接開啟管理頁面
