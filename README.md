# Wedding Butler V6.2 — 婚禮流程智慧圖示

本版由 V6.1 延伸，保留既有 Firebase／Firestore 資料結構。

## 新增
- 流程圖示選單新增「婚禮流程專用」分類。
- 包含下聘、回聘、坐高矮椅、戴戒指、見面禮、添妝、拜別、奉茶、迎娶、禮車、鳴炮、迎賓、禮金、入席、主持、進場、香檳塔、敬酒、合照、切蛋糕、抽捧花、遊戲、送客與撤場等圖示。
- 新增流程時，輸入流程名稱會即時自動建議對應圖示。
- 使用者手動修改圖示後，系統不會再覆蓋選擇。
- 修改既有流程時，預設保留原本圖示。

## 未加入

將所有檔案覆蓋至 GitHub Pages 專案即可；不需要修改 Firestore Rules。
