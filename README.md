# 婚禮管家 V5.3｜招待工作與桌次賓客分開版

## 工作名單
工作名單只顯示：

- 招待姓名
- 個人工作內容
- 備註

負責桌次不再顯示在工作名單成員卡片中，避免工作內容與服務桌次混在一起。

## 我的行程
招待登入後，「我的負責桌次」會直接顯示：

- 桌號
- 桌名
- 人數
- 賓客關係
- 桌次注意事項
- 完整賓客名單

點開桌次即可查看所有賓客；也可進一步開啟完整婚宴桌次資料。

## 工作內容
工作內容仍由任務或工作名單獨立管理，不會和桌次賓客資料混在一起。

## 總招待
總招待的「我的招待」卡片仍可查看每位招待的分工與負責桌次。
