# 婚禮管家 V3.0.7｜首頁可新增任務版

## 修正
首頁「任務」原本只能查看，現在新增：
- ＋新增任務
- ＋新增物品

新增時會自動把目前下拉選單中的「使用者」帶入負責人。

例如目前使用者為「紹廷」：
- 按新增任務，預設負責人為紹廷
- 按新增物品，預設負責人為紹廷
- 儲存後會立即出現在該使用者首頁

只有管理模式可以新增、修改或刪除。一般使用者仍只能查看與勾選完成。
