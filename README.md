# Wedding Butler V6.0｜總招待管理重構版

本版直接以使用者提供的 V5.4 專案修改。

## 總招待／爐叔
- 首頁顯示「總招待管理」
- 顯示全部招待與負責桌數
- 點招待展開工作內容及備註
- 負責桌次以 Badge 顯示
- 點桌次於原畫面查看桌名、關係、注意事項與賓客
- 不包含工作完成率

## 顯示條件
1. 工作名單有指定目前使用者為總招待；或
2. 使用者姓名為「爐叔」；或
3. 人員角色含「總招待」、「招待長」或「總召」。

當沒有明確指定總招待、但登入者符合上述角色時，會自動顯示名稱含「招待」的工作名單。

一般招待的「我的負責桌次」及其他功能維持原樣。Firebase rules 不需修改。
