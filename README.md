# 婚禮管家 V3.0.11｜多位負責人＋任務時間地點版

## 多位負責人
任務與準備內容的「DRI／負責人」改成可複選。

例如同一項準備內容可同時指定：
- 秀鳳伯母
- 紹廷
- 筱彤

每位被指定的人，都會在自己的「我的行程」看到同一筆任務或準備物品。

## 相容既有資料
既有單一負責人資料不需重建。
系統會同時支援：
- 舊欄位 owner
- 新欄位 owners

## 任務時間與地點
任務表單保留：
- 開始時間
- 結束時間
- 地點
- 地址
- Google 地圖

填寫後，「我的行程 → 任務」會顯示時間、地點、地址與「開啟 Google 地圖」。

將 ZIP 內全部檔案覆蓋上傳 GitHub Repository 根目錄。
