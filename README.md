# Wedding Control Center v15：流程收合修正版

## 修正內容

### 1. 流程事件現在可以明確收合
「08:00 文定物品清點」整個標題區都可以點擊：

- 點一下：收合女方準備、男方準備、額外準備
- 再點一下：重新展開
- 左側箭頭也可以操作
- 標題會顯示「點擊展開／點擊收合」

### 2. 不再顯示「未分組」標題
沒有指定婚禮階段的流程會直接顯示，不再額外包一層「未分組」。

若要把流程整理到文定、迎娶或婚宴：
1. 進入管理模式
2. 修改該流程
3. 選擇婚禮階段

### 3. 一般使用者看不到管理按鈕
一般模式會隱藏：

- 修改
- 刪除
- 上移
- 下移
- 加確認項
- 新增
- 設定

其他人仍可查看內容、勾選完成、展開／收合及開啟 Google 地圖。

## Firebase
沒有新增 collection，也不需要修改 Firestore rules。
